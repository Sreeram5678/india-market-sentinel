# India Market Sentinel

An intelligent, local-first financial dashboard that gives Indian retail investors an institutional-grade edge.

## Project Structure

- `.agent/skills/`: Specialized task definitions for the AI agent.
- `.agent/workflows/`: Step-by-step guides for building and maintaining the sentinel.
- `src/`: Core logic for filing analysis and sentiment auditing.
- `app.py`: Streamlit-based correlation dashboard.

## Getting Started

Run the build workflow:
`/build-sentinel`

AI-powered innovative financial analytics tool for Indian Markets using NLP and OCR.
